\section{Definición del sistema}
\subsection{Determinación del alcance del sistema}
Según el Pliego de Condiciones Técnicas y el análisis del sistema, se han identificado los siguientes casos de uso:
	\begin{enumerate}
	\item \textbf{Envío de documentos:} En este caso de uso se van a incluir todas las posibles acciones relacionadas con el envío de documentos por parte de un cliente, por lo que este caso de uso se subdivide en tres casos diferentes. Las diferencias entre los tres casos de uso están sólo en las acciones internas realizadas para cada tipo de documento.
		\begin{enumerate}
		\item \textit{Envío de cartas:} Este caso de uso representa la funcionalidad del envío de cartas a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\item \textit{Envío de telegramas:} Este caso de uso representa la funcionalidad del envío de telegramas a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\item \textit{Envío de fax:} Este caso de uso representa la funcionalidad del envío de fax a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\end{enumerate}
	
	
	\end{enumerate}
