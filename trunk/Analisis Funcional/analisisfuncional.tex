\section{Definición del sistema}
\subsection{Determinación del alcance del sistema}
Según el Pliego de Condiciones Técnicas y el análisis del sistema, se han identificado los siguientes casos de uso:
\subsubsection{Envío}
	\begin{enumerate}
	\item \textbf{Envío de documentos:} En este caso de uso se van a incluir todas las posibles acciones relacionadas con el envío de documentos por parte de un cliente, por lo que este caso de uso se subdivide en tres casos diferentes. Las diferencias entre los tres casos de uso están sólo en las acciones internas realizadas para cada tipo de documento.
		\begin{enumerate}
		\item \textit{Envío de cartas:} Este caso de uso representa la funcionalidad del envío de cartas a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\item \textit{Envío de telegramas:} Este caso de uso representa la funcionalidad del envío de telegramas a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\item \textit{Envío de fax:} Este caso de uso representa la funcionalidad del envío de fax a través del portal. Comprende la acción desde que el usuario inicia un nuevo envío hasta que recibe la confirmación del mismo.
		\end{enumerate}
	
	\item \textbf{Envío de paquetes:} Incluye todas las posibles acciones relacionadas con el envío de paquetes por parte de un cliente. Su clasificación es la siguiente:
		\begin{enumerate}
		\item \textit{Envío de paquete urgente:} Representa la funcionalidad del envío de paquetes urgentes.
		\item \textit{Envío de paquete nacional:} Representa la funcionalidad del envío de paquetes dentro del territorio nacional.
		\item \textit{Representa la funcionalidad del envío de paquetes fuera del área nacional}
		\end{enumerate}
	\end{enumerate}
	
\subsubsection{Filatelia}
	\begin{enumerate}
	\item \textbf{Bienvenida:} Caso de uso que se realiza cuando el usuario entra al sistema de filatelia.
	\item \textbf{Registro:} Cuando un usuario no es registrado debe completar este caso de uso para continuar en el portal filatélico.
	\item \textbf{Apadrinamiento:} Funcionalidad de poder apadrinar a un nuevo usuario del portal filatélico.
	\item \textbf{Colecciones:} Funcionalidad de gestión y visualización de colecciones filatélicas(sellos, rodillos, matasellos...).
	\item \textbf{Gestión online:} Funcionalidad de compra online de productos filatélicos.
	\item \textbf{At. Cliente:} Funcionalidad de atención al cliente del portal filatélico.
	\item \textbf{Asociaciones:} Funcionalidad de gestión de asociaciones filatélicas.
	\item \textbf{Puntos de venta:} Funcionalidad de gestión de puntos de venta de productos filatélicos.
	\end{enumerate}
	
\subsubsection{Información}
	\begin{enumerate}
	\item \textbf{At. Cliente:} Posee la funcionalidad de atención genérica al cliente.
	\item \textbf{Información corporativa:} Funcionalidad de información corporativa de la empresa.
	\end{enumerate}
	
\subsubsection{Gestión de usuario}
	\begin{enumerate}
	\item \textbf{Login:} Funcionalidad de inicio de sesión de usuario
	\item \textbf{Logout:} Funcionalidad de final de sesión de usuario
	\item \textbf{Registro:} Funcionalidad de registro de un nuevo usuario
	\end{enumerate}
